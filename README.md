# gtd-thing
translates the custom morse code in gtd (100% not ai trust me)         ---
record a audio through obs or roblox (roblox SUCKS you have to be in first person sitting right next to it) then send it into a morse code translator (https://morsefm.com/ works best)                                      ---
then put the translated morse code into the gui and it'll translate it for you           --
if you dont have python, download 3.14 from this website: https://www.python.org/downloads/, then you can open the cool.py file                                            ---
Notes if its not working: make sure you have opened the cool.py file and not python, make sure you have a good audio file for morse-fm, and make sure you're pasting in the morse code from the morse-fm website, not the translated junk
